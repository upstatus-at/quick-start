Upstatus Helm Repository

This is the official Helm repository for Upstatus, a Kubernetes monitoring and error analysis solution.

Adding the Helm Repository

To add this Helm repository to your Helm client:

helm repo add upstatus https://upstatus-at.github.io/quick-start/repo
helm repo update

Available Charts

  • upstatus - A Kubernetes monitoring and error analysis solution

Installing Charts

To install the Upstatus chart:

helm install upstatus upstatus/upstatus --namespace upstatus-system --create-namespace

Documentation

For more information about the Upstatus Helm chart, please refer to the GitHub repository.
